# Convert Voxel to World Coordinates

Convert a 3xN matrix of voxel coordinates to world coordinates with
respect to a given minc file.

## Usage

``` r
mincConvertVoxelMatrix(filename, voxel_matrix)
```

## Arguments

- filename:

  The minc file indicating a coordinate grid

- voxel_matrix:

  a 3xN matrix of voxel coordinates

## Value

a 3xN matrix of world coordinates
