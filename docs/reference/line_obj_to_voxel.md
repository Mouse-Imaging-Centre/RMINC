# Convert Lines to Voxel Coordinates

Convert a `bic_lines` object to world coordinates for plotting

## Usage

``` r
line_obj_to_voxel(line_obj, minc_file)
```

## Arguments

- line_obj:

  The `bic_lines` object of interest

- minc_file:

  The reference file for computing voxel coordinates

## Value

A `bic_lines` object in voxel coordinates
