# Voxel Vector

Given a voxel coordinate, return the value at those coordinates from a
set of minc files as a numeric vector

## Usage

``` r
mincGetVector(filenames, v1, v2, v3, v.length = NULL)
```

## Arguments

- filenames:

  Character vector with paths to minc files

- v1:

  First voxel coordinate

- v2:

  Second voxel coordinate

- v3:

  Third voxel coordinate

- v.length:

  The number of values to return
