# Object to Graph

Convert a bic_obj object to a graph for handy graph algorithms

## Usage

``` r
obj_to_graph(bic_obj)
```

## Arguments

- bic_obj:

  The `bic_obj` object, generated by
  [read_obj](https://mouse-imaging-centre.github.io/RMINC/dev/reference/read_obj.md)
  to be converted

## Value

an `igraph` object representing the object's mesh
