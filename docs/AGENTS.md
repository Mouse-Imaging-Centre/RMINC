# AGENTS.md — RMINC

## What This Is

R package for statistical analysis of MINC neuroimaging files. Compiled
C/C++ backend via Rcpp + autoconf.

## Build & Install

The package requires external MINC libraries (minc-toolkit-v2 or libminc
built from source) and HDF5. `configure.ac` locates them and generates
`src/Makevars` from `src/Makevars.in`.

``` sh
R CMD build .                          # creates RMINC_*.tar.gz
R CMD INSTALL RMINC_*.tar.gz           # install locally
```

If `configure` can’t find MINC, set `MINC_TOOLKIT` or
`--with-build-path=` to the install prefix.

## Testing

``` sh
Rscript -e "library(RMINC); devtools::test('.')"                          # all tests
Rscript -e "library(RMINC); devtools::test('.', filter = 'mincLm')"       # filtered
```

Tests depend on fixture data in `tests/testthat/rminctestdata/` and
`tests/testthat/RMINC-test-data-main/`.

## Docker-Based Testing

``` sh
docker build -t rminc-test .
docker run -v $(pwd):/RMINC rminc-test              # check + tests
docker run -v $(pwd):/RMINC rminc-test test          # tests only
docker run -v $(pwd):/RMINC rminc-test test mincLm   # filtered tests
```

## R CMD Check

``` sh
R CMD check --as-cran --no-install RMINC_*.tar.gz
```

## Documentation

NAMESPACE and man/ are generated by roxygen2 (v7.3.3). After editing R
source with roxygen comments:

``` sh
Rscript -e "roxygen2::roxygenise()"
```

## Architecture

- **R/** — Pure R code. Key files: `minc_voxel_statistics.R`,
  `minc_lmer.R`, `minc_interface.R`, `minc_FDR.R`,
  `minc_hierarchical_anatomy.R`, `minc_vertex_statistics.R`
- **src/** — C/C++ sources. Rcpp bindings in `RcppExports.cpp`
  (auto-generated), core I/O in `minc_reader.c`/`minc2_read.c`,
  modelling in `modelling_functions.c`, vertex TFCE in `vertex_tfce.cpp`
- **tests/testthat/** — testthat tests (28 test files)
- **inst/** — installed extras including documentation and autoconf m4
  macros

## Conventions

- Default branch is `develop` (CI triggers on push/PR to develop)
- R code uses 2-space indentation (see `.Rproj`)
- No Windows support (`OS_type: unix`)
- Requires R \>= 4.0
- CI matrix: ubuntu-latest + macos-latest, R versions: devel, release,
  oldrel-1, oldrel-2
- `src/Makevars` is generated — do not edit directly; edit
  `src/Makevars.in` instead
- `RcppExports.R` and `RcppExports.cpp` are auto-generated by Rcpp — do
  not edit
